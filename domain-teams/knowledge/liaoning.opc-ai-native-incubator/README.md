## 团队研究主线

- **OPC**：One Person Company，**AI 原生**小微主体形态（工具链、工作流、合规边界写清）。
- **政府合作**：**主谈与履约对接以沈阳市为主**；省级（辽宁）政策与资金工具可作为补充语境，须在文中区分层级与适用主体。与平台公司、园区、人才与就业政策工具相衔接的**可检验**叙述（工具、主体、里程碑），禁止空泛口号。
- **多种 OPC 业态**：IP 与 AIGC 内容仅是 playbook 之一；可扩展咨询、设计、数据服务、垂直 SaaS、电商等，每条 playbook 须写**准入标准、交付物、风险点**。
- **东北 / 辽宁叙事**：对标其他省市做法时，须写清**可迁移条件**与**本地化假设**，不得把外地政策当辽宁已生效事实。
- **首选落地城市**：平台公司意向为 **沈阳**（若变更须在 `inputs/` 与 WV 摘要中同步）。
