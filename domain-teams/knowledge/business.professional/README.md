# knowledge：business.professional

本目录为 **`business.professional`（专业商业与投融资）** 团队的唯一允许事实根之一（见 `_manifest/teams.json` 的 `search_roots`）。

## 放什么

- 口径约定：收入确认、ARR/MRR、毛利率、CAC/LTV 等定义与边界。
- 方法论骨架：市场 sizing、竞争图谱、尽调问题清单、路演叙事结构（不含未经验证的项目数据）。
- 行业片段：仅当可标注来源与适用边界时收录；**不得**把泛化结论伪装成客户项目事实。

## 禁止

- 将其他 `team_id` 的 WV 路径（如文商旅、数字文创）混作本团队检索依据。
- 存放密钥、未脱敏合同全文或个人隐私材料。
